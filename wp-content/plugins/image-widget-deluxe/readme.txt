Image Widget Deluxe is an easy to use widget plugin that allows you to change display order of the fields.

== Description ==

Image Widget Deluxe is a widget plugin that gives you that image widget we always seems to miss. Built with "Drag and Drop" interface so you easily can change the order of how the content should be displayed in the widget.
It integrates smoothly with WordPress media gallery so you can use existing media or swiftly upload new images.

[youtube https://www.youtube.com/watch?v=YsyLACwQhRY]

= Display what you need =
Image Widget Deluxe only displays fields that have values which means that if you don't need to display a title, caption or button - you just leave these fields empty.

== Installation ==

= Minimum Requirements =

* WordPress 3.8 or greater
* PHP version 5.2.4 or greater

= Easy installation =

Easy automatic installation by letting WordPress handle your files. Image Widget Deluxe can be installed easily with a  log in to your WordPress dashboard and then navigating to the Plugins menu to click Add New.

Search for the plugin by typing "Image Widget Deluxe" and click Search Plugins. Once youâ€™ve found Image Widget Deluxe plugin you can view details about it or simply install it by clicking "Install now".

Remember to enable the plugin.

= Manual installation =

The manual installation process is a bit more advanced for installing WordPress plugins. You have to have access to the WordPress installation. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Whenever there is an update pending for the plugin a notification will be visible in your WordPress dashboard. Before you update remember to backup your site just in case.
